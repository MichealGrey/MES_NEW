## 一、工厂概况

### 1.1 基本信息

| 项目 | 说明 |
|------|------|
| 工厂类型 | 半导体封测厂 (OSAT - Outsourced Semiconductor Assembly and Test) |
| 员工规模 | 约 2000 人 |
| 客户数量 | 约 200 家 |
| 产能规模 | 月产 5-8 亿颗芯片 |
| 运营模式 | 7x24 小时连续生产，3班2运转 |
| 业务范围 | 晶圆接收 → 减薄切割 → 贴片 → 引线键合 → 塑封 → 电镀 → 打标切筋 → 测试 → 终包出货 |

### 1.2 主要封装类型

- QFN/QFP (Quad Flat No-lead / Quad Flat Package)
- BGA (Ball Grid Array)
- QFN (Quad Flat No-lead)
- SOP (Small Outline Package)
- TO (Transistor Outline)
- DFN/DFN (Dual Flat No-lead)

### 1.3 主要测试类型

- CP 测试 (Chip Probing) - 晶圆级电性测试
- FT 测试 (Final Test) - 成品测试
- HTOL 高温寿命测试 - 可靠性测试
- 功能测试 (Functional Test)

## 三、生产部详细结构

### 3.1 生产部概览

生产部是OSAT核心部门，按工艺段划分为多条产线，实行 **3班2运转（Day/Night）**，覆盖7x24小时生产。

```
生产总监 Manufacturing Director (1人)
├── 生产副总监 Deputy Director (2人)
│   ├── 前段生产经理 Front-End Manager (1人) ─── 晶圆级封装
│   └── 后段生产经理 Back-End Manager (1人) ─── 测试与终包
│
├── 晶圆减薄切割车间 Wafer Grinding & Dicing (~220人)
├── 贴片车间 Die Attach (~250人)
├── 引线键合车间 Wire Bonding (~280人)
├── 塑封车间 Molding (~180人)
├── 电镀车间 Plating (~100人)
├── 打标切筋车间 Marking & Trim/Form (~120人)
├── 测试车间 Testing (~200人)
├── 终包车间 Final Pack (~80人)
├── 物料管理 Material Handling (~50人)
└── 生产支持 Manufacturing Support (~35人)
```

### 10.3 正向追溯 Forward Traceability

**触发场景：** 原材料异常、工艺变更、客户投诉

**使用角色：**

- 质量工程师：追踪异常物料影响范围
- CQE：向客户提供影响批次清单
- 计划员：调整受影响工单

**追溯路径：**

```
原材料批次 → 投入工单 → 生产批次 → 在制品/成品批次 → 出货记录 → 客户
```

### 10.4 反向追溯 Backward Traceability

**触发场景：** 成品不良、客户退货、良率异常

**使用角色：**

- 产品工程师：查找不良根因
- 工艺工程师：分析工艺参数
- 设备工程师：检查设备状态

**追溯路径：**

```
成品批次 → 生产工单 → 工艺参数 → 设备状态 → 操作人员 → 原材料批次
```

### 10.5 基因追溯 Genealogy Traceability

**触发场景：** 多层级产品追溯、晶圆级追溯

**使用角色：**

- 产品工程师：分析晶圆级良率
- 客户质量：提供完整追溯报告
- 工艺工程师：跨工艺段分析

**追溯路径：**

```
晶圆批次 → 减薄切割 → 贴片 → 键合 → 塑封 → 测试 → 终包 → 成品
         ↓           ↓       ↓       ↓       ↓       ↓      ↓
      工单记录    工单记录  工单记录  工单记录  工单记录  工单记录  出货记录
      设备状态    设备状态  设备状态  设备状态  设备状态  设备状态  包装记录
      工艺参数    工艺参数  工艺参数  工艺参数  工艺参数  工艺参数
      操作人员    操作人员  操作人员  操作人员  操作人员  操作人员
```

### 11.2 MES系统设计建议

| 设计要点 | 说明 |
|---------|------|
| **角色化工作台** | 不同角色登录显示不同工作台，突出核心功能 |
| **权限精细化** | 按角色+数据范围+操作类型三维权限控制 |
| **移动端优先** | 生产现场以平板/PDA为主，PC端用于配置和分析 |
| **实时性要求** | 生产数据延迟<1秒，看板刷新<3秒 |
| **防错设计** | 关键操作二次确认、条码校验、参数范围限制 |
| **追溯完整** | 支持正向/反向/基因三种追溯，响应时间<3秒 |
| **报表灵活** | 支持自定义报表、多维度分析、数据导出 |
| **系统集成** | 与ERP、QMS、EAP、WMS等系统无缝集成 |
