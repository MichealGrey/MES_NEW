## 一、项目背景

### 1.1 业务需求

半导体封测厂（OSAT）在生产过程中，设备会产生大量**毫秒级时序数据**，例如：

- **WireBond（焊线机）**：每打一根线采集 Bond Force、超声波功率、温度等，频率 1-10ms
- **DieBonder（固晶机）**：贴装瞬间的压力、位置、温度曲线，频率 5-50ms
- **CP/FT Tester（测试机）**：每个 Pin 的电压/电流/时序测试，频率 1-10ms
- **AOI（光学检测）**：高速相机拍照、图像处理，频率 10-100ms

这些**高频原始数据**的价值在于：
1. **实时监控**：设备状态、工艺参数、良率趋势
2. **质量管控**：SPC 分析、异常检测、自动 Hold
3. **预测维护**：设备故障预警、参数漂移检测
4. **工艺优化**：参数推荐、良率提升

### 1.2 现有问题

传统 MES 系统在处理设备数据时面临以下挑战：

| 问题 | 说明 | 影响 |
|------|------|------|
| **数据量大** | 单台 WireBond 设备每天产生 1000 万+ 条数据 | 关系型数据库性能瓶颈 |
| **实时性要求高** | 需要在秒级内完成数据采集、聚合、告警 | 传统批处理无法满足 |
| **分析需求复杂** | SPC、OEE、良率趋势、预测性维护 | 需要 OLAP 引擎支撑 |
| **系统耦合度高** | 数据采集、处理、存储、展示混在一起 | 难以扩展和维护 |

### 1.3 目标

设计一个**独立的设备数据平台**，与 MES 系统解耦，提供：

- ✅ 毫秒级数据采集与实时处理
- ✅ 灵活的 ODPS 四层数据架构
- ✅ 高性能时序存储与 OLAP 分析
- ✅ 标准化的 API 接口供 MES 调用
- ✅ 可扩展的架构支持未来新增设备

```python
            exchange='',
```

### 3.2 DWD 层（明细数据层）

**定位**：对 ODS 层数据进行清洗、标准化、窗口聚合，形成明细数据。

#### 流计算引擎：Apache Flink

##### Flink 实时聚合任务

```java
// WireBond 数据实时聚合
public class WireBondAggregationJob {
    public static void main(String[] args) throws Exception {
        StreamExecutionEnvironment env = StreamExecutionEnvironment.getExecutionEnvironment();
        env.setParallelism(4);
        
        // 1. 从 RabbitMQ 读取原始数据
        DataStream<BondRawData> rawStream = env
            .addSource(new RMQSource<>(
                "localhost", 5672, "equipment.raw.bond",
                new SimpleStringSchema()
            ))
            .map(json -> parseBondData(json))
            .filter(data -> data.isValid());
        
        // 2. 100ms 滚动窗口聚合
        DataStream<BondAggregation> agg100ms = rawStream
            .keyBy(BondRawData::getEquipmentId)
            .window(TumblingProcessingTimeWindows.of(Time.milliseconds(100)))
            .process(new BondAggregationProcessFunction());
        
        // 3. 写入 TDengine（DWD 层）
        agg100ms.addSink(new TDengineSink<>(
            "jdbc:TAOS://localhost:6030/mes_dwd",
            "INSERT INTO agg_bond_1s VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?)"
        ));
        
        // 4. SPC 实时计算
        DataStream<SpMetrics> spcStream = agg100ms
            .keyBy(SpMetrics::getEquipmentId)
            .process(new SpcCalculationFunction());
        
        // 5. 写入 ClickHouse（DWS 层）
        spcStream.addSink(new ClickHouseSink<>(
            "jdbc:clickhouse://localhost:8123/mes_dws",
            "INSERT INTO dws_equipment_kpi VALUES"
        ));
        
        // 6. CEP：连续超规检测
        Pattern<BondRawData, ?> pattern = Pattern.<BondRawData>begin("first")
            .where(new SimpleCondition<>() {
                public boolean filter(BondRawData value) {
                    return value.isOutOfSpec();
                }
            })
            .times(5).consecutive();
        
        PatternStream<BondRawData> patternStream = CEP
            .pattern(rawStream.keyBy(x -> x.equipmentId), pattern);
        
        DataStream<AlarmEvent> alarms = patternStream
            .select(new AlarmSelectFunction());
        
        // 7. 告警写入 RabbitMQ
        alarms.addSink(new RMQSink<>(
            "localhost", 5672, "equipment.alarm"
        ));
        
        env.execute("WireBond Real-time Aggregation");
    }
}

// 聚合处理函数
public class BondAggregationProcessFunction 
    extends ProcessWindowFunction<BondRawData, BondAggregation, String, TimeWindow> {
    
    @Override
    public void process(String key, Context ctx, 
                       Iterable<BondRawData> elements, Collector<BondAggregation> out) {
        double sum = 0, sumSq = 0, min = Double.MAX_VALUE, max = Double.MIN_VALUE;
        int count = 0, outOfSpec = 0;
        
        for (BondRawData data : elements) {
            double force = data.getBondForce();
            sum += force;
            sumSq += force * force;
            min = Math.min(min, force);
            max = Math.max(max, force);
            count++;
            
            if (force < 45.5 || force > 46.5) {
                outOfSpec++;
            }
        }
        
        double avg = sum / count;
        double std = Math.sqrt(sumSq / count - avg * avg);
        double cpk = calculateCpk(avg, std, 45.5, 46.5);
        
        out.collect(new BondAggregation(
            new Timestamp(ctx.window().getStart()),
            key,
            elements.iterator().next().getLotId(),
            count, avg, std, min, max, cpk, outOfSpec
        ));
    }
}
```

### 3.3 DWS 层（汇总数据层）

**定位**：面向业务场景的汇总数据，支持快速 OLAP 查询。

#### 存储引擎：ClickHouse

```sql
-- 设备 KPI 宽表（实时写入）
CREATE TABLE dws_equipment_kpi (
  time_bucket DateTime CODEC(Delta, ZSTD),
  equipment_id String,
  equipment_group LowCardinality(String),
  product_id LowCardinality(String),
  lot_id String,
  bond_count UInt32,
  avg_force Float32,
  std_force Float32,
  min_force Float32,
  max_force Float32,
  cpk Float32,
  yield_rate Float32,
  oee Float32,
  uptime_seconds UInt32,
  alarm_count UInt16,
  hold_count UInt16
) ENGINE = MergeTree()
PARTITION BY toYYYYMMDD(time_bucket)
ORDER BY (equipment_id, time_bucket)
TTL time_bucket + INTERVAL 90 DAY;

-- 物化视图：按小时自动聚合
CREATE MATERIALIZED VIEW dws_equipment_kpi_hourly
ENGINE = SummingMergeTree()
PARTITION BY toYYYYMMDD(hour_bucket)
ORDER BY (equipment_id, hour_bucket)
AS SELECT
  toStartOfHour(time_bucket) AS hour_bucket,
  equipment_id,
  equipment_group,
  product_id,
  sum(bond_count) AS total_bond_count,
  avg(avg_force) AS hourly_avg_force,
  avg(cpk) AS hourly_avg_cpk,
  avg(yield_rate) AS hourly_yield,
  sum(alarm_count) AS total_alarms
FROM dws_equipment_kpi
GROUP BY hour_bucket, equipment_id, equipment_group, product_id;

-- 良率汇总表
CREATE TABLE dws_yield_summary (
  time_bucket DateTime,
  equipment_id String,
  product_id String,
  lot_id String,
  step_code String,
  input_qty UInt32,
  pass_qty UInt32,
  fail_qty UInt32,
  yield_rate Float32,
  rework_qty UInt32,
  scrap_qty UInt32
) ENGINE = MergeTree()
PARTITION BY toYYYYMMDD(time_bucket)
ORDER BY (time_bucket, equipment_id, lot_id);
```

#### 典型查询场景

```sql
-- 1. 设备 OEE 趋势（最近 24 小时，1 分钟粒度）
SELECT 
  time_bucket,
  equipment_id,
  oee,
  uptime_seconds,
  yield_rate
FROM dws_equipment_kpi
WHERE equipment_id = 'WB-03'
  AND time_bucket > now() - INTERVAL 24 HOUR
ORDER BY time_bucket;

-- 2. 良率对比分析（按产品/设备，最近 7 天）
SELECT 
  product_id,
  equipment_id,
  avg(yield_rate) AS avg_yield,
  min(yield_rate) AS min_yield,
  max(yield_rate) AS max_yield,
  count() AS lot_count
FROM dws_equipment_kpi
WHERE time_bucket > now() - INTERVAL 7 DAY
GROUP BY product_id, equipment_id
ORDER BY avg_yield DESC;

-- 3. SPC 分析：Cpk 趋势（最近 30 天）
SELECT 
  toStartOfHour(time_bucket) AS hour,
  avg(cpk) AS avg_cpk,
  min(cpk) AS min_cpk,
  countIf(cpk < 1.33) AS low_cpk_count
FROM dws_equipment_kpi
WHERE equipment_id = 'WB-03'
  AND time_bucket > now() - INTERVAL 30 DAY
GROUP BY hour
ORDER BY hour;

-- 4. 设备告警统计（按类型/严重度）
SELECT 
  alarm_type,
  severity,
  count() AS alarm_count,
  countIf(severity = 'Critical') AS critical_count
FROM dws_equipment_kpi
WHERE equipment_id = 'WB-03'
  AND time_bucket > now() - INTERVAL 24 HOUR
GROUP BY alarm_type, severity;
```

```python
from pyspark.sql import SparkSession

# 预测性维护：设备故障预测
from pyspark.ml.feature import VectorAssembler
from pyspark.ml.regression import RandomForestRegressor

def predictive_maintenance():
    # 读取历史数据
    df = spark.read.parquet("hdfs:///mes-data/equipment-history")
    
    # 特征工程
    assembler = VectorAssembler(inputCols=[
        "avg_force", "std_force", "cpk", "uptime_hours",
        "alarm_count_7d", "pm_hours_ago"
    ], outputCol="features")
    
    # 训练随机森林模型
    rf = RandomForestRegressor(
        featuresCol="features",
        labelCol="hours_to_failure",
        numTrees=100
    )
    
    model = assembler.transform(df).fit(rf)
    
    # 预测当前设备状态
    current_data = spark.read.jdbc("jdbc:clickhouse://...", "dws_equipment_kpi")
    predictions = model.transform(assembler.transform(current_data))
    
    # 写入预警结果
    predictions.filter(col("hours_to_failure") < 24) \
        .write.jdbc("jdbc:mysql://...", "ads_maintenance_warning", mode="append")
```

### 4.1 WireBond 设备模拟器

```python
"""
WireBond 设备模拟器
模拟真实焊线机的毫秒级数据采集
"""

import pika
import json
import time
import random
import threading
from datetime import datetime
from typing import Dict, List

class WireBondSimulator:
    def __init__(self, equipment_id: str, rabbitmq_url: str):
        self.equipment_id = equipment_id
        self.rabbitmq_url = rabbitmq_url
        self.is_running = False
        self.bond_index = 0
        self.current_lot = None
        self.equipment_status = "Idle"
        
        # 设备参数配置
        self.config = {
            "bond_force": {"target": 46.0, "std": 0.15, "spec_lower": 45.5, "spec_upper": 46.5},
            "ultrasonic_power": {"target": 0.8, "std": 0.02, "spec_lower": 0.75, "spec_upper": 0.85},
            "bond_temp": {"target": 150.0, "std": 0.5, "spec_lower": 148.0, "spec_upper": 152.0},
            "bond_time": {"target": 0.5, "std": 0.02, "spec_lower": 0.45, "spec_upper": 0.55},
        }
        
        # 异常注入配置
        self.fault_injection = {
            "force_drift": False,       # Bond Force 漂移
            "high_variance": False,     # 高方差
            "continuous_out_of_spec": False,  # 连续超规
        }
    
    def start(self, lot_id: str = "LOT-2026001"):
        """启动模拟器"""
        self.is_running = True
        self.current_lot = lot_id
        self.equipment_status = "Processing"
        
        # 连接 RabbitMQ
        self.connection = pika.BlockingConnection(
            pika.URLParameters(self.rabbitmq_url)
        )
        self.channel = self.connection.channel()
        self.channel.queue_declare(queue='equipment.raw.bond', durable=True)
        
        # 启动数据采集线程
        self.data_thread = threading.Thread(target=self._collect_data)
        self.data_thread.start()
    
    def _collect_data(self):
        """数据采集主循环（10ms 间隔）"""
        while self.is_running:
            # 生成 Bond 数据
            data = self._generate_bond_data()
            
            # 发送到 RabbitMQ
            self._send_data(data)
            
            # 模拟异常注入
            self._inject_faults()
            
            self.bond_index += 1
            time.sleep(0.01)  # 10ms
    
    def _generate_bond_data(self) -> Dict:
        """生成单根线的 Bond 数据"""
        config = self.config
        
        bond_force = random.gauss(
            config["bond_force"]["target"],
            config["bond_force"]["std"]
        )
        
        return {
            "equipmentId": self.equipment_id,
            "equipmentType": "WireBonder",
            "lotId": self.current_lot,
            "bondIndex": self.bond_index,
            "timestamp": datetime.utcnow().isoformat(),
            "status": self.equipment_status,
            "params": {
                "bondForce": {
                    "value": round(bond_force, 4),
                    "unit": "N",
                    "spec": [config["bond_force"]["spec_lower"], 
                             config["bond_force"]["spec_upper"]]
                },
                "ultrasonicPower": {
                    "value": round(random.gauss(
                        config["ultrasonic_power"]["target"],
                        config["ultrasonic_power"]["std"]
                    ), 4),
                    "unit": "W",
                    "spec": [config["ultrasonic_power"]["spec_lower"], 
                             config["ultrasonic_power"]["spec_upper"]]
                },
                "bondTemp": {
                    "value": round(random.gauss(
                        config["bond_temp"]["target"],
                        config["bond_temp"]["std"]
                    ), 2),
                    "unit": "°C",
                    "spec": [config["bond_temp"]["spec_lower"], 
                             config["bond_temp"]["spec_upper"]]
                }
            },
            "quality": {
                "wirePull": {
                    "value": round(random.gauss(8.5, 0.3), 2),
                    "unit": "g",
                    "spec": [8.0, None]
                }
            },
            "materials": {
                "wireLotId": "GW-20260520-001",
                "capillaryId": "CAP-WB-001"
            }
        }
    
    def _inject_faults(self):
        """异常注入（模拟设备故障/参数漂移）"""
        if self.fault_injection["force_drift"]:
            # Bond Force 缓慢漂移
            self.config["bond_force"]["target"] -= 0.01
        
        if self.fault_injection["high_variance"]:
            # 方差增大
            self.config["bond_force"]["std"] *= 1.1
    
    def _send_data(self, data: Dict):
        """发送数据到 RabbitMQ"""
        self.channel.basic_publish(
            exchange='',
            routing_key='equipment.raw.bond',
            body=json.dumps(data),
            properties=pika.BasicProperties(
                delivery_mode=2,  # 持久化
                content_type='application/json'
            )
        )
    
    def stop(self):
        """停止模拟器"""
        self.is_running = False
        if self.data_thread:
            self.data_thread.join()
        if self.connection:
            self.connection.close()
```

```python
            {"name": "Propagation_Delay", "unit": "ns", "target": 5.0, "std": 0.3, "spec": [None, 6.0]},
        ]
    
    def start(self, lot_id: str = "LOT-2026001"):
        self.is_running = True
        self.current_lot = lot_id
        
        self.connection = pika.BlockingConnection(
            pika.URLParameters(self.rabbitmq_url)
        )
        self.channel = self.connection.channel()
        self.channel.queue_declare(queue='equipment.raw.test', durable=True)
        
        self.data_thread = threading.Thread(target=self._collect_data)
        self.data_thread.start()
    
    def _collect_data(self):
        while self.is_running:
            # 模拟测试一颗芯片（100-500ms）
            test_data = self._generate_test_data()
            
            self.channel.basic_publish(
                exchange='',
                routing_key='equipment.raw.test',
                body=json.dumps(test_data),
                properties=pika.BasicProperties(
                    delivery_mode=2,
                    content_type='application/json'
                )
            )
            
            self.test_count += 1
            time.sleep(random.uniform(0.1, 0.5))  # 100-500ms
    
    def _generate_test_data(self) -> Dict:
        """生成单颗芯片的测试数据"""
        bin_distribution = {1: 0.96, 2: 0.03, 3: 0.01}  # Bin 分布
        
        test_results = []
        for item in self.test_items:
            value = random.gauss(item["target"], item["std"])
            
            # 判断是否超规
            spec_lower = item["spec"][0]
            spec_upper = item["spec"][1]
            is_out_of_spec = False
            
            if spec_lower and value < spec_lower:
                is_out_of_spec = True
            if spec_upper and value > spec_upper:
                is_out_of_spec = True
            
            # 根据超规情况决定 Bin
            if is_out_of_spec:
                bin_result = random.choices([2, 3], weights=[0.75, 0.25])[0]
            else:
                bin_result = 1
            
            test_results.append({
                "testItem": item["name"],
                "value": round(value, 4),
                "unit": item["unit"],
                "specLower": spec_lower,
                "specUpper": spec_upper,
                "isOutOfSpec": is_out_of_spec
            })
        
        return {
            "equipmentId": self.equipment_id,
            "equipmentType": "FTTester",
            "lotId": self.current_lot,
            "testIndex": self.test_count,
            "timestamp": datetime.utcnow().isoformat(),
            "testProgram": {
                "id": "FT-BGA-7nm-v1.0",
                "revision": "R3"
            },
            "hardware": {
                "loadboard": "LB-BGA-001",
                "socket": "SKT-2026-001"
            },
            "testResults": test_results,
            "binResult": bin_result,
            "environment": {
                "temperature": round(random.gauss(25.0, 0.2), 1),
                "humidity": round(random.gauss(45.0, 1.0), 1)
            }
        }
    
    def stop(self):
        self.is_running = False
        if self.data_thread:
            self.data_thread.join()
        if self.connection:
            self.connection.close()
```

## 八、关键技术决策

### Decision 1：为什么选择 TDengine 而不是 InfluxDB？

**选择**：TDengine

**理由**：
1. 国产开源，社区活跃，文档完善
2. 原生支持超级表（类似物化视图）
3. 写入性能优异（单表 10 万+ TPS）
4. 内置时间窗口函数，减少应用层计算
5. 压缩率高（节省存储空间）

### Decision 2：为什么选择 Flink 而不是 Spark Streaming？

**选择**：Flink

**理由**：
1. 真正的流处理（Spark Streaming 是微批）
2. 延迟更低（毫秒级 vs 秒级）
3. CEP 支持完善（复杂事件处理）
4. 状态管理强大（Exactly-Once 语义）
5. 适合实时聚合场景

### Decision 3：为什么选择 ClickHouse 而不是 MySQL 做 OLAP？

**选择**：ClickHouse

**理由**：
1. 列式存储，聚合查询快 10-100 倍
2. 支持物化视图（自动聚合）
3. 压缩率高（节省 5-10 倍存储）
4. 适合宽表设计
5. 与 Grafana 集成完善
